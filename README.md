# Steam Monitoring Script

Скрипт в фоне отслеживает загрузки Steam и выводит отчёт в консоль каждую минуту. Параметры мониторинга настраиваются.

## Что умеет скрипт

1. **Считывание скорости загрузки** по логам Steam.
2. **Вывод в консоль скорости загрузки игры каждую 1 минуту (на протяжении 5 минут)**
3. **Учитывание статуса загрузки** например, если загрузка на паузе.
4. **Вывод названия игры** какая именно игра сейчас скачивается.

Примечание: статус загрузок берётся из логов Steam, путь к Steam из реестра Windows.

## Алгоритм работы

1. Находит папку Steam в реестре Windows или по стандартным путям.
2. Читает "хвост" лога `content_log.txt` (последние 2 MB) и парсит: прогресс загрузки, статус (Downloading/Suspended), скорость из строки `Current download rate: X Mbps`.
3. Каждую минуту (5 раз): читает новые строки лога с последней позиции, перечитывает "хвост" (512 KB) для актуальной скорости, обновляет состояние и выводит отчёт.
4. В отчёт попадают только игры в загрузке или на паузе; название берётся из `appmanifest_<AppID>.acf` во всех библиотеках Steam.

## Для чего полезен

- Проверить скорость загрузки без открытия клиента Steam.
- Убедиться, что загрузка идёт или стоит на паузе (например, при ограничении трафика).
- Получить список и названия скачиваемых игр в одном месте (консоль/лог).
- Основа для доработки: экспорт в файл, уведомления, мониторинг с другого ПК по сети.

## Запуск

```bash
python main.py
```

Требуется Python 3. Скрипт рассчитан на Windows (поиск Steam через реестр).

## Структура

- `main.py` : точка входа
- `steam.py` : поиск пути Steam (реестр и стандартные пути), название игры по AppID
- `monitor.py` : чтение `content_log.txt`, парсинг, отчёты, настройки мониторинга
